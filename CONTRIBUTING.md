# Contributing

Contributions welcome!

## How to Contribute

1. Fork repository
2. Create feature branch
3. Commit changes
4. Push to branch
5. Create Pull Request

## Development Setup

```bash
git clone https://github.com/Amirezamky9/hermes-toolkit.git
cd hermes-toolkit
pip install -r requirements.txt
```

## Code Style

- Follow PEP 8
- Add docstrings
- Add type hints
- Write tests

## Testing

```bash
# Run tests
pytest

# Run with coverage
pytest --cov=.
```

## Questions?

Open an issue on GitHub.
