# Pattern: Scheduling + Notification

## تشخیص سریع — چه بپرسی تا بفهمی Schedule هست؟
- [ ] کاربر گفت "هر روز / هفتگی / ماهیانه / هر X ساعت"؟
- [ ] "تایمر / زمان‌بندی / روتین / خودکار اجرا بشه"؟

## سوالات هدفمند (قبل از جستجو)
| بپرس | تا بفهمی |
|------|---------|
| چند وقت یك بار اجرا بشه؟ (دقیقه / ساعت / روز / هفته / ماه) | Interval یا Cron |
| چه روزا و ساعتی؟ منطقه زمانی خاصی مدنظره؟ | Schedule Trigger config |
| اگه شرطی برقرار باشه اجرا بشه؟ (مثلاً اگه دیتا جدید بود) | نیاز به IF/Condition |
| بعد از اجرا چیکار کنه؟ (اعلان / ذخیره / API) | خروجی نهایی |
| چجوری اطلاع بده؟ (ایمیل / اسلک / تلگرام / هیچی) | Notification channel |
| حجم دیتایی که پردازش میکنه چقدره؟ | نیاز به Batch یا نه |

## کلمات کلیدی برای جستجوی مشابه
`cron job n8n`, `daily report`, `scheduled notification`, `automated email`

## Credentialهایی که باید چک کنی
SMTP / Slack / Telegram / Twilio / API مقصد
