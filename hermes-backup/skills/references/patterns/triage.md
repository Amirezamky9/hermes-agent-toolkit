# Pattern: Triage

## تشخیص سریع — چه بپرسی تا بفهمی Triage هست؟
- [ ] کاربر گفت "دسته‌بندی / مسیریابی / دسته‌بندی خودکار / sort"؟
- [ ] "ورودی رو بخونه و بفرسته مسیر مختلف"؟

## سوالات هدفمند (قبل از جستجو)
| بپرس | تا بفهمی |
|------|---------|
| چی رو میخوای دسته‌بندی کنی؟ (ایمیل / تیکت / پیام / دیتا) | Input type |
| چند دسته داری؟ | تعداد مسیرهای Switch |
| Rule-based یا AI؟ (قوانین ثابت یا تحلیل متن) | نیاز به LLM یا نه |
| اگه به هیچ دسته‌ای نخورد چکار کنه؟ | Fallback path |
| خروجی هر دسته چیه؟ (ایمیل / اسلک / شیت / API) | Destination |

## کلمات کلیدی برای جستجوی مشابه
`triage automation`, `auto classify`, `email routing`, `support ticket n8n`

## Credentialهایی که باید چک کنی
بستگی به خروجی داره: Slack / Email / Google Sheets / CRM API
