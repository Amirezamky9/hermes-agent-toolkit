# Pattern: Chatbot

## تشخیص سریع — چه بپرسی تا بفهمی Chatbot هست؟
- [ ] کاربر گفت "ربات / ربات چت / تلگرام / اسلک / واتساپ"؟
- [ ] "پیام بده و جواب بگیره" یا "مکالمه داشته باشه"؟

## سوالات هدفمند (قبل از جستجو)
| بپرس | تا بفهمی |
|------|---------|
| چه پیام‌رسانی؟ (تلگرام / اسلک / واتساپ) | نوع Trigger و Send |
| توکن ربات داری؟ | Credential مورد نیاز |
| با AI جواب بده یا جواب ثابت؟ | نیاز به LLM یا نه |
| فقط چت کنه یا کار دیگه‌ای هم بکنه؟ | Scope پروژه |
| حافظه داشته باشه مکالمات قبلی رو یادش بمونه؟ | نیاز به Memory Node |
| چت تکی یا گروهی؟ | Session Key strategy |

## کلمات کلیدی برای جستجوی مشابه
`telegram bot`, `slack chatbot`, `n8n telegram ai`, `chatbot with memory`

## Credentialهایی که باید چک کنی
Telegram API / Slack API / OpenAI (اگه AI داره)
