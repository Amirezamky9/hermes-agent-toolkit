# Hermes Built-in Webhook Platform

Quick reference for creating and exposing Hermes webhook routes on this server.

## Subscribing (create a route)

Inside the Hermes WebUI container, `hermes` is at `/app/venv/bin/hermes` (not in PATH):

```bash
export PATH=$PATH:/app/venv/bin
hermes webhook subscribe <name> \
  --prompt "Your prompt with {payload.fields}" \
  --description "What it does" \
  --deliver origin
```

Returns URL + auto-generated HMAC secret.

## Testing

```bash
PAYLOAD='{"message":"test"}'
SECRET="<auto-generated-or-custom-secret>"
SIG=$(echo -n "$PAYLOAD" | openssl dgst -sha256 -hmac "$SECRET" | awk '{print $2}')

curl -X POST https://<tunnel-or-domain>/webhooks/<name> \
  -H "Content-Type: application/json" \
  -H "X-Hub-Signature-256: sha256=$SIG" \
  -d "$PAYLOAD"
```

Response: `{"status": "accepted", "route": "<name>", "delivery_id": "..."}`

## Listing / Removing

```bash
hermes webhook list
hermes webhook remove <name>
```

## Port

Always 8644 — defined in `config.yaml` under `platforms.webhook.extra.port`.

## Exposing externally (this server)

This server has Traefik on ports 80/443 (Docker, `network_mode: host`). The webhook platform (port 8644) runs natively outside Docker. Two approaches:

### A) Traefik file provider (recommended for permanent route)

Add to `/docker/traefik/dynamic.yml`:
```yaml
http:
  routers:
    webhook-gateway:
      rule: "Host(`webhook.YOUR_DOMAIN`)"
      entryPoints: ["websecure"]
      tls: { certResolver: letsencrypt }
      service: webhook-gateway
  services:
    webhook-gateway:
      loadBalancer:
        servers:
          - url: "http://127.0.0.1:8644"
```

Requires mounting the file + adding `--providers.file.filename=/dynamic.yml` to Traefik's docker-compose command.

### B) Cloudflare Quick Tunnel (temporary / debug only)

```bash
export PATH=$PATH:$HOME/.local/bin
cloudflared tunnel --url http://localhost:8644
# Wait ~15-20s for URL: https://<random>.trycloudflare.com
```

## ⚠️ Pitfalls

- **HMAC only** — this platform does NOT accept Bearer tokens. CacheCat and browser extensions won't work here (use port 9999 FastAPI instead).
- **`hermes` CLI not in PATH inside container** — use `PATH=$PATH:/app/venv/bin` every time.
- **TryCloudflare rate-limit** — if `cloudflared` fails with "invalid UUID length: 0", wait 10-15s and retry.
